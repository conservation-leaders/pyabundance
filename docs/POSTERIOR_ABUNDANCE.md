# Posterior Abundance / Ranef-Like Outputs

v0.7 adds empirical-Bayes latent abundance summaries for fitted pcount models.

For each site, pyabundance computes the conditional distribution:

```text
P(N_i = n | y_i, theta_hat) ∝ P(N_i = n | theta_hat) * product_j P(y_ij | N_i = n, theta_hat)
```

The support is `n = max(y_i), ..., K`; missing visits are skipped. These distributions condition on fitted model parameters and do **not** include uncertainty in beta, detection coefficients, NB `r`, or ZIP `psi`. Intervals can therefore understate total uncertainty. Use parametric bootstrap with refitting when broader uncertainty matters.

## Predicted lambda vs posterior N

- `predict_lambda()` is the covariate-based expected latent abundance for the abundance component.
- `posterior_mean` is the site-level expected `N_i` after conditioning on observed counts at that site.
- `observed_max_count` is the largest observed count at the site and is the lower bound of posterior support.

These quantities are related but not interchangeable.

## Ranef-like summaries

```python
from pyabundance import load_example_pcount, pcount_df

data = load_example_pcount("poisson")
fit = pcount_df(
    site_data=data.site_data,
    count_cols=data.count_cols,
    abundance_formula=data.abundance_formula,
    detection_formula=data.detection_formula,
    mixture="poisson",
    K=data.K,
)

ranef = fit.ranef()
print(ranef.head())
```

`fit.ranef()` is an alias for `fit.posterior_abundance_summary()`.

Summary columns include posterior mean, mode, median, interval bounds, posterior SD, probability at the observed maximum count, probability above the observed maximum count, and zero-abundance probabilities.

## Full posterior abundance matrix

```python
support, probs = fit.posterior_abundance(return_support=True)
```

`probs` has shape `(n_sites, K + 1)`. Rows sum to one, and probabilities below each site's observed maximum count are zero.

## Long and wide DataFrames

```python
fit.posterior_abundance_dataframe()
fit.posterior_abundance_dataframe(long=True)
fit.abundance_dataframe(include_posterior=True)
```

The long format has one row per site × abundance support value. This can be large for large `K` or many sites.

## Posterior abundance samples and total abundance

```python
samples = fit.posterior_abundance_samples(nsim=1000, seed=123)
total = fit.total_abundance_posterior(nsim=1000, seed=123)
print(total.summary())
```

Samples have shape `(nsim, n_sites)`. Total abundance summaries are generated by summing one latent abundance draw per site for each simulation.

## Posterior predictive checks

```python
counts = fit.posterior_predictive_counts(nsim=100, seed=123)
check = fit.posterior_predictive_check(statistic="zero_count", nsim=100, seed=123)
print(check)
```

Supported posterior predictive check statistics:

- `"sse"`
- `"total_count"`
- `"zero_count"`
- `"max_count"`

This is different from parametric bootstrap: posterior predictive counts here condition on fitted parameters and posterior `N_i` draws, while parametric bootstrap simulates from the fitted model and can refit each replicate.

## Reporting

```python
fit.export_summary("report.md", include_posterior_abundance=True)
```

Posterior abundance is excluded from reports by default to avoid large output. When included, reports contain total abundance summaries, the first 10 site summaries, and a caution that these distributions condition on fitted parameters.
