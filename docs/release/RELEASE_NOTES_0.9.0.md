# pyabundance 0.9.0 Release Notes

v0.9.0 is an external-alpha release rehearsal. It does not add a new ecological likelihood family and does not change validated pcount likelihood formulas.

## Highlights

- Guarded TestPyPI Trusted Publishing workflow.
- Guarded draft real-PyPI workflow requiring maintainer approval.
- Cross-platform maturin wheel matrix and smoke tests.
- TestPyPI install smoke workflow.
- MkDocs deployment workflow.
- Benchmark artifact workflow with optional R parity input.
- API freeze review and public API tests.
- External alpha reviewer guide, issue templates, and PR template.
- Distribution metadata checks and wheel/sdist smoke scripts.
- Conservative coverage threshold of 80%.

## Clean-room note

No R/GPL source code was copied, inspected, translated, or paraphrased. R remains a black-box comparison target only.

## Status

This release is intended for external alpha review and TestPyPI rehearsal, not a final v1.0 production release.
