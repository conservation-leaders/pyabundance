# pyabundance

pyabundance is a clean-room Python package for ecological abundance modelling with a Rust numerical core.

The current pcount family supports Poisson, negative-binomial, and zero-inflated Poisson N-mixture models with:

- matrix-first and pandas/formula APIs;
- uncertainty and diagnostics;
- model-selection and reporting helpers;
- posterior abundance / ranef-like summaries.

Use the tutorial and API pages in this documentation as the contributor-facing reference for v0.8 release rehearsals.
